## Codebase Patterns
- Models use SQLAlchemy `DeclarativeBase` with `Mapped` type annotations and UUID string PKs
- No Alembic migrations — new tables auto-created by `Base.metadata.create_all()`, new columns added via idempotent `ALTER TABLE` in `init_db()`
- All models must be imported in `init_db()` (database.py) AND exported from `models/__init__.py`
- Use `IF NOT EXISTS` pattern (try/except) for column migrations in `init_db()`
- JSON columns use `sqlalchemy.JSON` with `default=list` or `default=dict`
- Timestamps use `DateTime` with `default=lambda: datetime.now(timezone.utc)`
- Run `python -m py_compile` from `backend/` dir for quick typecheck; use `python -c "from app.models.X import Y"` to verify imports
- Git operations must be run from repo root `/mnt/c/Users/dobec/actions/`
- Backend Python runs from `/mnt/c/Users/dobec/actions/backend/`
